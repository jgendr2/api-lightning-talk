First we need to understand WTF Is Happening When I Click a Button on a website

So you click a button to view this web page: "https://www.avantcredit.com/customer/dashboard"

You just made a request to our web server!


Rails receives this request and decides what to do next.

We need information about the customer who just requested this page . . aka you!


So we do a few queries in the database to get the necessary customer info

Now we have to give you something to view in your web browser


So Rails uses this customer information it just found in the database to generate HTML (HyperText Markup Language)

Your browser turns this:



Into what you see on a web page.
